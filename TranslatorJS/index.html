How to Use

  • Select the language you want to translate from.
  • Select the language you want to translate to.
  • Enter your text in the input box.
  • Click "Translate" to see the translation.
  • The translation and custom script will appear below.

Tips

  • Some words won't work unless you use the base word. So instead of "afterwards" try "after"
